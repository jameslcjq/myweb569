**老九的工资系统**是一款面向学校工资业务的 Windows 桌面软件。它把工资基础数据、社保个税文件、月度工资报账、一体化系统对接、绩效工资、年初调整和统计报表放在同一个工作台里，减少重复整理 Excel、反复核对数据和手工制作导入文件的时间。

工资业务最麻烦的地方，往往不是某一张表，而是“数据来源多、口径要统一、结果还要能导入系统”。这款软件的目标就是把这些分散步骤串起来，让每月工资报账更清楚、更稳定。

## 适合哪些场景

- 学校、单位每月需要处理在职工资、退休工资、遗补或其他工资数据。
- 需要把工资表、社保、个税、公积金等数据统一整理后报账。
- 需要生成保险导入、凭证导入、工资导入、补发工资等一体化系统文件。
- 需要做绩效工资差异测算、年初社保个税调整、工资年报和统计分析。
- 希望保留工资报账历史，方便后续查询、重新生成、月结归档和备份恢复。

## 主要功能

### 工资数据集中管理

软件支持维护在职工资、退休工资、其他工资等基础数据，可通过 Excel 导入、一体化导出文件导入和手工编辑等方式更新。本地数据会作为月度报账、绩效工资、年初调整和统计报表的基础。

关键人员匹配以证件号码等稳定字段为核心，减少同名、重名或文件名判断带来的误差。

### 月度工资报账

月度工资模块可以结合工资表、社保未申报汇总、个税计算表和单位基础设置，生成报账所需的阶段结果和最终结果。

常见输出包括：

- 保险导入文件
- 凭证导入文件
- 工资导入文件
- 补发工资文件
- 报表预览和历史报账记录

软件会对当前月份、社保所属期、个税所属期、人员匹配和数据差异进行校验，尽量把问题拦在生成之前。

### 一体化对接

软件内置一体化系统页面入口，支持多标签页使用，并承接工资导出、预算下载、保险推送、凭证推送、工资导入、补发工资导入、指标匹配等流程。

推送前会检查当前登录单位和目标单位，保险、凭证等任务按队列串行执行，某一步失败时会停止后续步骤，避免错误继续扩散。

### 绩效工资和年初调整

绩效工资模块可以对比两个期间的工资数据，识别晋级、小档、退休、调出、新进、调入等变化，生成绩效工资差异结果。

年初调整模块可以结合工资表、公积金账号表、社保明细、个税模板和社保基数模板，生成工资导入、住房申报、个税导入和社保基数导入等结果。

### 年报、统计和日常维护

软件还提供工资年报、统计分析、乡镇补贴、退休房补、人事信息一致性检查、邮件附件下载、数据备份和恢复等功能，适合把日常工资业务沉淀成一个长期可维护的工作台。

## 使用建议

第一次使用时，建议先完成单位设置，再导入基础工资数据。之后每月按“导入工资表、导入社保个税、生成报账结果、推送一体化、月结归档”的顺序处理。

如果某个月数据还没补齐，可以先生成阶段结果用于核对；社保、个税和工资表确认无误后，再生成最终报账结果。
